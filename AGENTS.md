## Target Process API

Only use GET operations.

### Authentication

Target process uses HTTP basic authentication.

### Overview

https://www.ibm.com/docs/en/targetprocess/tp-dev-hub/saas?topic=v1-getting-started

### API Reference

https://md5.tpondemand.com/api/v1/index/meta

## Ruby

All Ruby related commands should be invoked through the
`~/.config/opencode/ruby_cmd.sh` script. Example: `
~/.config/opencode/ruby_cmd.sh ruby --version` and
`~/.config/opencode/ruby_cmd.sh rails --help`
